# O Poder do Intestino - Landing (ebook 21 dias)

Landing page de venda do ebook da Katiane Nogueira. Single-file HTML puro (CSS inline), sem build.

- index.html - pagina completa (hero, sintomas, plano 21 dias, pricing, FAQ, autora, CTA). Inclui Meta Pixel + InitiateCheckout.
- mockup_ebook.png - mockup do ebook.

## Deploy
Publicado na Hostinger: https://ebook21dias.poderdointestino.com (via lftp, a partir da VPS).
Sem build - basta subir index.html (e imagens) para a pasta publica do subdominio.

Historico: versao anterior em React/Vite descontinuada (problemas de hash/cache). Refeito como HTML puro em 2026-05.
